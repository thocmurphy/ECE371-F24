### boardBlink.py
 - Blinks the Pico LTE's on board LED

 - Taken from Pico LTE SDK
 - Sends HTTP GET request to server in config.json

### post.py
 - Taken from Pico LTE SDK
 - Sends HTTP POST request to server in config.json

### put.py
 - Taken from Pico LTE SDK
 - Sends HTTP PUT request to server in config.json
